# Trello Reminder - Project Brief

## Goal
Build a personal work reminder system that syncs with Trello:
- Pull boards/lists/cards via Trello API
- Generate an overview of what needs attention "now"
- Later: send reminders via channels (Telegram/Email/Push) and schedule checks

## Current status
Paused at: fetching data from Trello API and presenting an overview.

## Assumptions
- Single user (personal) at first, multi-user optional later
- Prefer simple, auditable architecture
- Documentation-first (design before coding)

- No UI implementation yet
- No database decisions yet
- No deployment decisions yet
